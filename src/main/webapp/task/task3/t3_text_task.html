This task with computation number of people by class. "titanic_data.csv" file contains "pclass" column that contains "1", "2" or "3" number of class.

You task is to build pie chart the same as the pie chart on lower window on right. Result of your action you can see in top window. Pie chart that you should build shows number of people depending on the class.

For check you solution you can press the "check" button. After that you will see the result of check near with the button. That's all about task. Good luck!
